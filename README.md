Navmarg Consulting Website (Single-File SPA)

This package contains the complete, standalone code for the Navmarg Consulting website, implemented as a single-file Static Page Application (SPA) using HTML, vanilla JavaScript, and Tailwind CSS.

How to Run the Website

Since the entire application is contained in one file (index.html), it can be run directly in any modern web browser without needing a web server or complex setup.

Save the Code: Copy the entire content of the index.html file provided above.

Create the File: Open a plain text editor (like Notepad, VS Code, Sublime Text, etc.).

Paste and Save: Paste the copied code into the editor. Save the file as index.html. Ensure the file extension is .html.

Open in Browser: Locate the saved index.html file on your computer and double-click it. It will open automatically in your default web browser (Chrome, Firefox, Edge, etc.).

Key Features

Single-File: All HTML, CSS (via Tailwind CDN), and JavaScript are in one file.

Responsive Design: Optimized for mobile, tablet, and desktop viewports.

Client-Side Routing: The site behaves like a multi-page application using vanilla JavaScript to switch between content sections instantly.

Recruitment Messaging: The "Careers" page includes the newly updated value propositions for recruitment.
